# ts-redux-saga-ssr

typescript & redux & redux-saga & server side rendering

## How to run

```
$ npm install
$ npm start
$ open http://localhost:3000/
$ # see console
```

## How to develop

```
$ npm install
$ npm run watch
$ # VS Code `Debug: Run` (F5)
```
